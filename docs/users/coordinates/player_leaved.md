## player_leaved

> **Сообщение отправляется со стороны backend**

> Данное сообщение отправляется все пользователям после того как один из пользователей вышел из игры/websocket закрыл соединение.

## Структура сообщения:

```json
{
  "type": "player_leaveed",
  "user_id": 2
}
```

## Описание полей

- **user_id** — ID пользователя.
